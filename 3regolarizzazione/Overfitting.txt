Affrontiamo il problema dell'overfitting

BIAS: Misura quanto sono distanti le nostre predizioni dai valori corretti

VARIANZA: Misura la differenza nelle predizioni utilizzando diversi datasets,
    ci dice quanto il modello è sensibile alla casualità dei dati nel set di addestramento

TRADEOFF BIAS/VARIANZA
    BIAS e VARIANZA sono altamente correlati:
    - alto bias alta varianza (male)
    - basso bias alta varianza
    - alto bias bassa varianza 
    - basso bias bassa varianza (bene)

    Il nostro lavoro è quello di riuscire a trovare la giusta combinazione 
    per ottenere bias e varianza che siano entrambi più piccoli possibile.

UNDERFITTING: bassa varianza e alto bias 

OVERFITTING: basso bias e alta varianza (modello troppo complesso)

Riconoscere l'overfitting:
confrontare l'errore sul set di addestramento con l'errore sul set di test,
se l'errore sul set di test è notevolmente maggiore dell'errore sul test di addestramento,
allora molto probabilmente il nostro modello soffre di overfitting.
Un gap eccessivo tra errore di addestramento ed errore di test è un indicatore di overfitting.

SOLUZIONI:

- Diminuire la complessità del modello rimuovendo alcune proprietà,
    ad esempio diminuire il grado del polinomio.

- Penalizzare i pesi utilizzando la regolarizzazione (CONSIGLIATO).
